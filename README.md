## Getting started

1. Clone this repo and enter!

   ```bash
   git clone https://github.com/danielhdezller/pasiona-DanielHernandez.git
   cd elit/
   ```

2. Install dependencies.

   ```bash
   cd travel-port-daniel-hernandez && npm i
   ```

3. Start the app.

- on the travel-port-daniel-hernandez folder run:
  ```bash
  npm start
  ```

4. Test.

- on the travel-port-daniel-hernandez folder run:

  ```bash
  npm test
  npm test:coverage
  npm test:watch
  ```
